# irfanarsyad.github.io

Personal portfolio website for Irfan Hukama Arsyad - Senior Programmer & Full Stack Developer.

## Live Demo

**[https://irfanarsyad.github.io](https://irfanarsyad.github.io)**

## Features

- Responsive design (mobile-friendly)
- Bilingual support (Indonesian & English)
- Dynamic GitHub projects (auto-fetch from GitHub API)
- Smooth scroll animations
- Interactive hover effects
- Animated particles background
- Dark mode skills section

## Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- GitHub Pages

## Sections

- **Hero** - Introduction with animated background
- **About** - Personal info and GitHub stats
- **Skills** - Tech stack with progress bars
- **Projects** - Auto-fetched from GitHub repositories
- **Contact** - Links to GitHub and email

## Development

```bash
# Install dependencies
npm install

# Build CSS
npx tailwindcss -i ./src/input.css -o ./src/output.css

# Watch mode (for development)
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
